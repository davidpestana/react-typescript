## Componentes

Todas las interfaces de usuario están hechas de distintas piezas. Nosotros vamos a describir cada
una de estas piezas como componentes. Los componentes nos permiten seguir el patrón de divide
y vencerás con el que podremos romper la aplicación en distintos componentes más pequeños y
reusables.

Los componentes de React se basan en los conceptos de entradas (props) y estados internos (states)
que veremos un poco más adelante.

Los componentes pueden ser de dos tipos:

- De clase

- Funcionales
