# 20. Fragments

[← Índice](README.md) | [← Anterior: Propiedad children](19-propiedad-children.md)

---

Agrupar varios elementos sin añadir un nodo al DOM. En TSX se usa `<></>` o `<Fragment>`.

```tsx
const Lista = () => (
  <>
    <li>A</li>
    <li>B</li>
    <li>C</li>
  </>
);
```
